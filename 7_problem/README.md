# 🧠 Impossible Data Science Problem

## 🎯 Objective
You are given a complex customer dataset. Your goal is to perform:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Hypothesis Testing
- Feature Engineering
- Statistical Testing

---

## 📊 Tasks

### 1. Data Preprocessing
- Handle missing values (simulate if needed)
- Detect and treat outliers
- Encode categorical variables

### 2. Exploratory Data Analysis (EDA)
- Distribution analysis
- Correlation analysis
- Detect skewness

### 3. Hypothesis Testing
Perform:
- Pearson Correlation Test
- Chi-Square Test
- Normality Test
- t-test / Mann-Whitney

### 4. Business Questions
- Does income affect spending score?
- Does gender impact churn?
- Is region related to purchase frequency?

---

## ⚠️ Challenge Rules
- Use proper statistical justification
- Use p-values for decision making
- Clearly define H0 and H1

---

## 🚀 Expected Output
- Clean dataset
- Visualizations
- Hypothesis conclusions
- Final insights report
